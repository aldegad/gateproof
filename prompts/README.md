# Gateproof Prompt Packs

These prompt files make report capture more repeatable.

Each prompt should:

- name the exact skill to use
- name the target
- request a markdown report
- encourage evidence-first language
- align with an eval case when possible

Current prompt pack:

- `prompts/demo-api/kisa-check.md`
- `prompts/demo-api/full-security-check.md`

These prompts are not mandatory, but they make report artifacts easier to compare over time.
