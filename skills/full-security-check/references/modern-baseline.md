# Modern Security Baseline

Use this baseline for practical security depth reviews.

## Primary frameworks

- OWASP Top 10
- OWASP ASVS
- NIST SSDF
- CIS-style hardening instincts where infra is visible

## Mandatory review themes

1. Authentication and session management
2. Authorization and privilege boundaries
3. Input validation and injection risk
4. Secret management
5. Sensitive data handling
6. Logging and detection quality
7. Dependency and supply-chain risk
8. CI/CD and deployment trust boundaries
9. Cloud and infrastructure exposure
10. Abuse resistance and operational resilience

## Severity model

- Critical
- High
- Medium
- Low

Prioritize by realistic exploit path, blast radius, and ease of abuse.
