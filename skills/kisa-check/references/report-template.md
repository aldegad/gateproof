# KISA Report Template Notes

## Suggested control buckets

- Identity and account management
- Access control
- Data protection
- Logging and monitoring
- Hardening and exposure reduction
- Patch and update hygiene
- Administrative or operational evidence

## Reporting style

- Keep findings concrete.
- Include exact files, endpoints, configs, or commands when possible.
- Call out whether the gap is:
  - implementation gap
  - configuration gap
  - evidence gap
