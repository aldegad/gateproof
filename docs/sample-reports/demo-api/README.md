# Demo API Sample Reports

This folder keeps durable sample deliverables for the risky `examples/demo-api/` target.

Included assets:

- `demo-api-security-report.ko.md`
  - Korean combined source report
- `demo-api-security-report.ko.pdf`
  - Korean combined shareable PDF
- `demo-api-kisa-report.ko.md`
  - Korean KISA-focused source report
- `demo-api-kisa-report.ko.pdf`
  - Korean KISA-focused shareable PDF
- `demo-api-full-security-report.ko.md`
  - Korean full-security source report
- `demo-api-full-security-report.ko.pdf`
  - Korean full-security shareable PDF
- `demo-api-findings.csv`
  - git-friendly findings matrix
- `demo-api-findings.xlsx`
  - Excel-friendly findings matrix

The `exports/` directory is intentionally not used for long-lived samples. Intermediate render artifacts such as HTML are excluded from this folder on purpose.
